#   docs:     Atualizar documentação

#   :recycle			mudança de código que não corrige um bug nem adiciona uma funcionalidade
#   :zap:				mudança de código que melhora o desempenho

#   :construction:		mudanças no processo de build ou ferramentas auxiliares e bibliotecas
